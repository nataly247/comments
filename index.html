Flutter and Dart

Flutter is a free and open-sourse UI Framework for building beautiful, natively compiled applications for mobile, web, and desktop from a single codebase. Flutter code is compiled to native ARM machine code using Dart's native compilers. You can use one programming language and codebase to create different apps.
